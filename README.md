# godot-parse-toml-plugin

适用于Godot4的Toml文件解析插件。

Toml file parsing plugin for Godot4.

# 计划 plan

1. 支持 json 转 toml

2. 支持 Unicode 字符 `\uXXXX` 或 `\UXXXXXXXX`
